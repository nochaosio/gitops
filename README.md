# GitOps Platform

One command brings up Argo CD, which installs the whole platform from Git.

## Setup (curl)

Prerequisites: `git`, `helm`, `kubectl`, and a kubeconfig pointing at the target cluster.

```bash
curl -fsSL https://raw.githubusercontent.com/nochaosio/gitops/main/bootstrap.sh | bash
```

## Per-environment setup (kind)

```bash
make vendor               # vendor upstream charts (once)
make kind-up ENV=dev      # create the kind cluster
make argocd-install ENV=dev
make bootstrap ENV=dev    # apply the root app-of-apps
make argocd-password      # initial admin password
make argocd-ui            # UI at https://localhost:8080 (user: admin)
```

## Tools

PostgreSQL (cloudnative-pg), Kafka (strimzi), Flink, Jaeger, OpenTelemetry,
kube-prometheus-stack, Gitea, cert-manager. Each one is an App generated by the
ApplicationSet from `platform/*/*/config.json`.
